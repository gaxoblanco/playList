
# Los contenedores corren en:
    Frontend ->  http://localhost:8000/
    Backend  ->  http://localhost:5000/
    PhpMyAdmin -> http://localhost:8080/
    
* Para desarrollo, levanta el servidor con:
     $env:ENVIRONMENT="development"; $env:NODE_ENV="development"; docker-compose up --build
     

* Para producción, levanta el servidor con:
    docker-compose up

* Para ver logs
    docker-compose logs -f


** Para detener ambos servicios
    docker-compose down

*** Limpiar todo
    docker system prune -a --volumes -f

*** Reconstruir el contenedor del front-end
    docker-compose down
    docker-compose build sora-playlist-frontend
docker-compose build sora-playlist-api
